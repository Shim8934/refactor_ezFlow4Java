사용자 환경설정 (IE 버전 8.0 이상)

그룹웨어의 기능을 정상적으로 사용하기 위해서는 아래와 같이 설정하여야 합니다.

ㆍOS : Windows 9X, Windows 2003 server, Windows 2000 Professional, Windows ME, Windows XP, Windows Vista, Windows 7
ㆍ웹브라우저 : IE 5.5 버전 (SP2) 이상(IE 6.0 버전 이상에서 최적화)
현재 사용중인 IE의 버전 확인하려면 IE를 실행한 후, 도움말 > Internet Explorer 정보 메뉴를 클릭합니다.



그룹웨어에 로그인하기 위해서는 인증서를 설치해야 합니다.

1. 그룹웨어 로그인 페이지에 처음 접속하거나, 인증서 파일을 설치 하지 않는 경우 ActiveX컨트롤 설치 확인창이 나타나며, 설치 버튼을 클릭하여 설치합니다.
1. 인증서는 한번만 설치하시면 다음 접속할 때부터는 나타나지 않습니다.

신뢰할 수 있는 사이트에 추가

1. IE -> 도구 -> 인터넷옵션 -> 보안 -> 신뢰할 수 있는 사이트 -> 사이트 클릭 ->
- http://ezekp.kaoni.com, http://sp.ezekp.kaoni.com, http://portal.ezekp.kaoni.com 입력
- https 체크버튼 해제

팝업 차단 설정

3. IE -> 도구 -> 인터넷옵션 -> 개인정보 탭 -> 팝업차단 -> 설정 -> *.ezekp.kaoni.com 입력

1. 웹브라우저 설정 즉, 신뢰할수 있는 사이트 추가 및 팝업 차단 설정을 하지 않을경우 컴포넌트파일이 설치되지 않아, 그룹웨어를 정상적으로 사용할수 없습니다.

보안수준 설정

IE -> 도구 -> 인터넷옵션 -> 보안 탭 -> [내 컴퓨터]에 있는 파일에서 액티브 콘텐츠가 실행되는 것을 허용

IE -> 도구 -> 인터넷옵션 -> 보안 탭 -> 사용자 지정수준 -> 크기 및 위치 제한 없이 스크립트 실행 창을 열 수 있습니다.



IE -> 도구 -> 인터넷옵션 -> 보안 탭 -> 설정 -> 스크립틀릿 허용



그룹웨어 최초 로그인 시 또는 시스템 버전이 변경되었을 경우 컴포넌트를 자동 다운로드 받습니다.



1. 그룹웨어 웹사이트가 팝업 차단 설정이 되어 있을 경우 자동 다운로드가 되지 않으므로 팝업 허용 설정을 하신 후 이용하시기 바랍니다.


2. 컴퍼넌트 다운로드는 필수 사항입니다. 컴퍼넌트가 정상적으로 설치되지 않을 경우, 다음과 같은 현상이 발생할수도 있습니다.
  1. 첨부파일 추가 아이콘 클릭 시 스크립트 오류발생
  2. 전자결재, 전자우편 내용입력 한 부분이 안 보임
  3. 그룹웨어 게시판, 문서함, 조직도 리스트 안 보임
  4. 새편지를 쓸때 내용 입력창이 안 보임
컴퍼넌트 다운로드 중 실패하였을 경우, 초기화면에서 새로고침을 실행하면 다시 다운로드 받기를 시작합니다.


 


IE -> 도구 -> 인터넷옵션 -> 일반 탭 -> 검색기록 -> 설정 -> 임시인터넷파일 -> 저장된 페이지의 새 버전 확인 -> '페이지를 열 때마다'로 선택

IE -> 도구 -> 인터넷옵션 -> 일반 탭 -> 검색기록 -> 설정 -> 임시인터넷파일 -> 사용할 디스크공간 -> 권장 용량의 최소 공간으로 설정
